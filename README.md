# 🎨 Draw It!

그림으로 소통하는 실시간 멀티플레이 웹 게임입니다.  
출제자는 그림을 그리고, 참여자는 그림을 보고 정답을 맞히거나 다양한 게임 모드를 통해 경쟁합니다.

---

## 📌 프로젝트 소개

**Draw It!** 은 WebSocket 기반의 실시간 통신을 활용하여  
여러 사용자가 동시에 참여할 수 있는 캐주얼 게임 플랫폼입니다.

- 실시간 그림 동기화
- 채팅 기반 정답 입력
- 투표 및 점수 집계 시스템
- 다양한 게임 모드 지원 (그림 맞히기, 끝말잇기 등)

---

## 🧩 주요 기능 흐름

### 1. 게임 시작 및 방 관리
- 닉네임을 입력한 뒤 방을 생성하거나 기존 방에 참여할 수 있습니다.
- 방 생성 시 게임 모드, 비밀번호 여부 등을 설정할 수 있습니다.
- 대기실에서는 현재 참여자 목록과 채팅을 통해 소통할 수 있습니다.

### 2. 대기실 & 실시간 채팅
- 방에 입장한 유저들은 게임 시작 전까지 대기 상태로 유지됩니다.
- 채팅 메시지는 사용자 캐릭터 옆 말풍선으로 실시간 표시됩니다.
- 방장은 게임을 시작할 수 있으며, 모든 참여자는 상태를 실시간으로 공유합니다.

### 3. 그림 맞히기 게임 진행
- 게임이 시작되면 출제자와 정답자에게 각각 다른 안내 모달이 표시됩니다.
- 출제자는 제시어를 받고 캔버스에 그림을 그립니다.
- 참여자들은 그림을 보고 채팅을 통해 정답을 입력합니다.
- 상단 타이머 프로그레스바를 통해 남은 시간이 실시간으로 표시됩니다.

### 4. 게임 종료 및 투표
- 라운드가 종료되면 그림에 대한 투표가 진행됩니다.
- 투표 시간과 진행 상태가 시각적으로 표시됩니다.
- 투표 결과에 따라 점수가 집계되며 순위가 결정됩니다.

### 5. 결과 및 명예의 전당
- 최종 순위와 점수가 결과 화면에 표시됩니다.
- 월별 인기 그림을 확인할 수 있는 **명예의 전당** 기능을 제공합니다.

---

## 🔄 추가 게임 모드 – 끝말잇기

- 그림 맞히기와 동일한 로비 UI에서 진행됩니다.
- 턴 기반으로 단어를 입력하며 제한 시간 내에 답변해야 합니다.
- 각 턴의 성공·실패 여부가 로그로 기록됩니다.
- 라운드 종료 시 점수와 결과가 정리되어 표시됩니다.
